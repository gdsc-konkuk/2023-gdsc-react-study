* '=='과 '==='의 차이는 무엇인가요? 
</br>언제 어느 연산자를 사용하는 것이 좋을까요? ==은 값만 비교한다.
</br>자료형을 고려하지 않는다
</br>예를 들어 문자 ‘5’와 숫자 5를 ‘5’==5 는 true라고 한다
</br> 형변환이 필요 없을때 사용한다
</br>
</br>  ===은 값 뿐만 아니라 자료형도 고려한다
</br>앞선 예제에서 ‘5’는 문자열이고 5는 숫자이기 때문에 ‘5’===5는 false이다
</br>정확한 일치 여부를 판단할때 사용한다   좋은 코딩 습관을 만들기 위해서는 ==보단 ===을 쓰는것이 좋다
